# ⛏️ Minecraft 數學歷史遊戲

俾 6 歲小朋友學數學嘅 Minecraft 風格冒險遊戲。

## 🎮 遊戲特色

- **Minecraft 風格** — 方塊造型、鑽石獎勵、秘密基地
- **數學練習** — 適合 6 歲嘅加減法、數數、比大小
- **歷史主題** — 每個關卡認識一個古文明（第1關：古埃及）
- **獎勵系統** — 答對攞鑽石，喺基地自由堆疊
- **iPhone 支援** — 可加入主畫面全螢幕玩

## 🕹️ 點樣玩

1. 㩒「開始遊戲」
2. 喺秘密基地㩒「古埃及金字塔」
3. 聽法老王講故事
4. 睇金字塔缺咗邊塊磚
5. 㩒啱答案 → 獲得鑽石 💎
6. 返基地睇你收集嘅鑽石！
